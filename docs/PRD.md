# Stablecoin Liquidity Monitor

## Product Goal
Provide a macro liquidity dashboard that tracks stablecoin supply and macro liquidity indicators to detect crypto liquidity regime shifts.

## Target Users
- crypto traders
- macro researchers
- onchain analysts

## Core Question
1. What is the current liquidity regime?
2. Is stablecoin liquidity expanding or contracting?
3. Are macro liquidity conditions improving?

## Key Signals
- Stablecoin supply
- Stablecoin net exchange flow
- Fed liquidity proxy
- Reverse repo (RRP)
- Treasury General Account (TGA)

## Core Outputs
- Current liquidity regime: Risk-On / Neutral / Risk-Off
- Short regime explanation based on macro + stablecoin conditions
- Historical trend charts for stablecoin supply and macro liquidity proxies
- Stablecoin breakdown table for supporting detail

## Product Scope (MVP)

The MVP dashboard should include:

- Liquidity regime summary
- Key stablecoin metrics
- Historical liquidity trend chart
- Stablecoin breakdown table

The MVP should NOT include:

- trading signals
- prediction models
- user accounts
